# EdkShellPkg

**EdkShellPkg** contains meta data files for building the
[EDK](EDK)
[Shell](Efi-shell).
EdkShellPkg has been retired in lieu of
[ShellPkg](ShellPkg) which
implements the UEFI Shell 2.x as specified by the UEFI Forum.
