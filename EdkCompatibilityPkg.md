# EdkCompatibilityPkg

EdkCompatibilityPkg provides header files and libraries that enable you
to build the [EDK](EDK) module
in UEFI 2.0 + Framework 0.9x mode. This package also provides Thunk
modules that enable Framework 0.9x modules to work with
[UEFI](UEFI) 2.3.1 / PI 1.2
[EDK II](http://www.tianocore.org/edk2/) Platforms.
