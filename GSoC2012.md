## Students

**Next Steps**

- Look over our <a href="Tasks" class="wikilink" title="Tasks">Tasks</a>
  page and find one (or more) projects that sound interesting to you.
- You **must** submit your project proposal for the tianocore
  organization on the [GSoC2012
  (google-melange.com)](http://www.google-melange.com/gsoc/homepage/google/gsoc2012)
  page before [April 6, 2012 at 19:00
  UTC](http://www.google-melange.com/gsoc/events/google/gsoc2012).
  - There is an application template below
  - The application deadline is **final**, so don't be late!
  - You can submit applications for more than one project if you find
    multiple projects interesting. Although, you can only be able to be
    assigned to a single project in the end.
- Please join our <a href="edk2-devel" class="wikilink"
  title="https://edk2.groups.io/g/devel"><span>https://edk2.groups.io/g/devel</span></a>
  email list and start asking about the project you are interested in
  - Let us know if you need more information!

**More information**

- Feel free to suggest you own project idea. (It is best to discuss your
  idea first on <a href="edk2-devel" class="wikilink"
  title="https://edk2.groups.io/g/devel"><span>https://edk2.groups.io/g/devel</span></a>.)
- Tianocore GSoC page:
  <http://www.google-melange.com/gsoc/org/google/gsoc2012/tianocore>

Note: We primarily use the
<a href="BSD_License" class="wikilink" title="BSD License">BSD
License</a>, and it is generally expected that your project will be
completed utilizing this license.

## Application Template

Students, please prepare a project proposal with the following
information:

- Contact information (email)
- Personal description, qualifications
  - skills
  - past projects/experiences
  - any previous open source contributions
- Explain your interest in UEFI or firmware
- Your plans for the project
  - high level plan
  - 1st draft timeline
- Will you be able to become a
  <a href="Member_Types#Basic_Contributor" class="wikilink"
  title="Basic Contributor">Basic Contributor</a> by agreeing to our
  <a href="Contribution_Agreement" class="wikilink"
  title="Contribution Agreement">Contribution Agreement</a>? (Just
  answer yes or no. If yes, then we would complete this process
  separately.)

This application/project proposal will need to be uploaded to the main
GSoC website between [March 26th and April
6th](http://www.google-melange.com/gsoc/events/google/gsoc2012). The
accepted applications will be announced on [April
23rd](http://www.google-melange.com/gsoc/events/google/gsoc2012).

To submit your application visit the Google Summer of Code site, and
submit your application with the tianocore project.

- <http://www.google-melange.com/gsoc/org/google/gsoc2012/tianocore>
