# Basetools

The BaseToolsPkg provides build related tools for both EDK and EDK2. Build related tools include AutoGen, Build, GenSec,
GenFV, GenFW, and GenRds (see [EDK II Tools List](edk_ii_tools_list.md)). For details refer to the document in
BaseTools/UserManuals directory.

The tools are developed under the [EDK II Build Tools](edk2_buildtools.md) project.
